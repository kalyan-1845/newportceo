## 🛠️ Quick Launch

### 1. Requirements
* Install [Node.js](https://nodejs.org/) (v18.0+).

### 2. Startup Command
```bash
git clone https://github.com/kalyan-1845/newportceo.git
cd newportceo
npm install
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.
